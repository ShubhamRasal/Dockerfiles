## Dockerfile

```dockerfile
FROM centos:latest
MAINTAINER  shubhamrasal

# Install httpd
RUN yum install httpd -y

# Copy file to document root
COPY . /var/www/html/

#expose port
EXPOSE 80

#start the httpd service
CMD ["/usr/sbin/httpd", "-D","FOREGROUND"]
```
